ZOOSHOP LR4 — инструкция по установке

1. Распакуйте архив.
2. Папку zooshop_lr4 скопируйте в htdocs:
   C:\xampp\htdocs\zooshop_lr4

3. Запустите Apache и MySQL в XAMPP / OpenServer.

4. Откройте сайт:
   http://localhost/zooshop_lr4/

5. База данных zooshop_lr4_db создается автоматически при первом открытии сайта.

6. Если хотите импортировать базу вручную:
   - откройте phpMyAdmin;
   - импортируйте файл database.sql;
   - затем откройте сайт.

7. Файл подключения:
   includes/config.php

8. Если в MySQL другой пользователь или пароль, поменяйте:
   $user = 'root';
   $password = '';

9. Для проверки:
   http://localhost/zooshop_lr4/test.php
